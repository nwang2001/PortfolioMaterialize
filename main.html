Materialize

What is Materialize?

It is a design language that combines principles of successful design along with innovation and technology. It was created and designed by Google with the goal of developing a system of design that allows for a unified user experience across all its products on any platform. It is also an UI component library created using HTML, CSS, and Javascript.

How to use it on your project?

There are multiple ways. You can download the file, either the Materialize standard version or the Sass version. You can also use CDN, NPM, or Bower to deliver, link, or fetch the files. CDN stands for content delivery/distribution network, NPM is Node Package Manager, and Bower is a package manager for client-side libraries and components.
